2. store 폴더에 calcStore.js 만들기

	import {create} from 'zustand'
	const calcStore = create( (set,get)=> ({}) )

	export default calcStore;

    주의할 점:
			create( 무명함수 ) 형태이다.
			create( (){} )

			그런데,
			(set,get) {} 하면 일반 함수정의
	        (set,get) { return {} } 하면 객체({})를 반환하는 함수

			간단히
			(set,get) => ({})

	즉, create함수는 
	set, get이라는 매개변수를 전달하고, 객체를 반환하는 함수를 만든다.
	set,get은 zustand에서 관리하는 메소드인데, 사용법은 실제 프로젝트를 보면 알게 됩니다.

	또 중요한 것은 create함수는 함수를 만들어서 반환하는 것이다.
	const calcStore = create( (set,get)=> ({}) )
	결국 calcStore는 함수이다. 그래서 calcStore() 식으로 실행한다.
